# 📝 Todo List Project

A simple and interactive Todo List web application built using HTML, CSS, and JavaScript.

## 🚀 Features

- ➕ Add new tasks
- ✅ Mark tasks as completed
- ⏳ Mark tasks as pending
- ❌ Delete tasks
- 🎨 Dynamic UI updates

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)
